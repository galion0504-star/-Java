Лабораторная работа №7: Многопоточность в Java

Описание

Многопоточность в Java позволяет выполнять несколько задач одновременно в рамках одного приложения. Это значительно увеличивает производительность при обработке больших объемов данных и позволяет эффективно распределять ресурсы процессора.

Ключевые инструменты и методы:

- Thread / Runnable — базовые способы создания и запуска потоков в Java

- synchronized — модификатор для обеспечения исключительного доступа к общему ресурсу

- join() — метод, заставляющий главный поток ожидать завершения работы дочернего потока

- ExecutorService — высокоуровневый интерфейс для управления пулом потоков и распределения задач

Содержание заданий:

1. Многопоточное вычисление суммы элементов массива

- Реализация параллельного сложения чисел с использованием двух отдельных потоков:

- Массив разделяется на две равные части, каждая из которых обрабатывается отдельным потоком

- Используются статические переменные sum1 и sum2 для хранения результатов каждого потока

- Главный поток агрегирует результаты вычислений после завершения работы обоих потоков

2. Поиск наибольшего элемента в матрице

Параллельный поиск максимального значения в двумерном массиве:

- Матрица делится на строки для независимой обработки рабочими потоками

- Каждый поток находит максимум в своей строке и сохраняет результат в массив rowMax

- Физическое сравнение локальных максимумов происходит в главном потоке программы

3. Симуляция работы склада (Вариант 1)

Многопоточная модель переноса товаров весом до 150 кг группой из 3 грузчиков:

- Product — класс товара с полями name (название) и weight (вес)

- Warehouse — склад с очередью товаров и общим весом currentWeight

- Loader — грузчик, который забирает товары со склада

- Синхронизация процесса погрузки и разгрузки при достижении лимита веса с помощью synchronized
