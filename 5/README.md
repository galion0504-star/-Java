Лабораторная работа №5: Регулярные выражения в Java

Описание

Работа посвящена изучению и практическому применению регулярных выражений (Regex). Регулярные выражения представляют собой мощный инструмент для поиска, проверки (валидации) и модификации текстовой информации на основе заданных шаблонов.

Основные инструменты:

В языке Java работа с Regex строится на двух основных классах пакета java.util.regex:

Pattern: Скомпилированное представление регулярного выражения.

Matcher: Движок, который выполняет операции сопоставления с текстом, используя объект Pattern.

Синтаксис и квантификаторы:

\d - Любая цифра

\w - Любая буква или цифра

\b - Граница слова

{n,m} - Повторение от n до m раз

[A-Z] - Набор символов (в данном случае — заглавные латинские буквы)

^ и $ - Начало и конец строки соответственно

Содержание заданий6

1. Поиск чисел в тексте (NumberFinder)

Программа извлекает все числовые значения (включая целые и дробные) из произвольного текста. Поддерживаются числа с разделителями как в виде точки, так и запятой.

2. Валидация пароля (PasswordValidator)
   
Реализована проверка пароля на соответствие строгим критериям безопасности:

- Длина от 8 до 16 символов

- Только латинские буквы и цифры

- Обязательное наличие минимум одной заглавной буквы и одной цифры

3. Исправление регистра (LowercaseUppercaseFinder)
   
Программа находит ошибки набора текста, где заглавная буква идет сразу за строчной, и выделяет их восклицательными знаками.

4. Проверка IP-адреса (IPAddressValidator)

Шаблон для проверки корректности IPv4-адреса:

- 4 блока чисел, разделенных точками

- Каждое число в диапазоне 0–255

5. Поиск слов по первой букве (WordFinderByLetter)

Гибкий поиск слов, начинающихся с определенного символа, игнорируя регистр.
