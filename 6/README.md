Лабораторная работа №6: Коллекции и дженерики в Java

Описание

Данная работа посвящена изучению каркаса коллекций (Java Collections Framework), механизмов обобщенного программирования (Generics) и способов эффективного перебора данных с помощью итераторов. Коллекции позволяют управлять группами объектов, обеспечивая удобные методы поиска, сортировки и модификации данных.

Основные интерфейсы коллекций:

Collection — базовый интерфейс для всех коллекций

- List — упорядоченный список, допускающий дубликаты (ArrayList, LinkedList)

- Set — коллекция уникальных элементов без строгого порядка (HashSet, TreeSet)

- Map — хранилище пар «ключ-значение» (HashMap, TreeMap)

Ключевые принципы:

- Generics — повышение безопасности типов за счет проверок на этапе компиляции

- Iterator — объект для последовательного и безопасного обхода элементов любой коллекции

- Comparable/Comparator — интерфейсы для настройки логики сравнения и сортировки объектов

Содержание заданий:

1. Подсчет частоты слов (TopWords)
   
Программа выполняет чтение текстового файла и анализ частоты встречаемости слов:

- Использование Map<String, Integer> для хранения пар «слово — количество повторений»

- Очистка слов от знаков препинания с помощью replaceAll("[^a-zа-я0-9]", "")

- Сортировка данных с помощью Comparator для вывода 10 самых популярных слов

2. Обобщенный стек (Stack<T>)

Реализация структуры данных «Стек» на основе массива с применением дженериков:

- Метод push — добавление элемента на вершину стека

- Метод pop — удаление и возврат верхнего элемента

- Метод peek — просмотр вершины без удаления элемента

- Метод size — возврат количества элементов

3. Учет продаж в магазине (Вариант 1: ArrayList, но в проекте HashMap)

Разработка системы мониторинга реализованных товаров:

- Использование ArrayList<Product> для хранения списка продаж

- Использование HashMap<String, Integer> для подсчёта количества продаж каждого товара

- Функционал включает вывод списка всех продаж, расчёт общей суммы и поиск самого популярного товара
