Лабораторная работа №1: Знакомство с Java и базовые алгоритмы Описание

Данная работа является вводной и направлена на освоение базового синтаксиса языка Java, работу с командной строкой, компилятором (javac) и виртуальной машиной Java (JVM). Структура проекта

В репозитории представлены две основные программы:

1. Primes.java — Программа для поиска простых чисел в диапазоне от 1 до 100.

Реализован метод isPrime(int num) для проверки числа на простоту.

Использованы циклы for и оператор остатка от деления %.

2. Palindrome.java — Анализатор строк на наличие свойств палиндрома.

Метод reverseString(String vord): переворачивает строку с помощью циклов и методов charAt().

Метод isPalindrome(String vord): сравнивает оригинальную строку с перевернутой, используя метод .equals().
